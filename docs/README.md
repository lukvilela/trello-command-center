## Como adaptar

1. Edite cada arquivo `0X-*.md` substituindo os placeholders
2. Mantenha a numeração e títulos pra preservar a navegação no dashboard
3. O dashboard renderiza estes arquivos automaticamente em `/docs`
4. Cross-links entre arquivos funcionam (use `[texto](02-arquitetura.md)`)
